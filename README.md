# 🧠 BrainDeck: AI-Powered Smart Flashcards

BrainDeck is a dynamic productivity and learning application designed to automate the study process. Instead of manually writing out study notes, users simply input a topic, and BrainDeck instantly generates a custom deck of interactive flashcards using AI. 

Built as part of a **3-Day Challenge Sprint**, this project focuses on a clean Minimum Viable Product (MVP) utilizing core web fundamentals and modern AI API integration.

## ✨ Core Features

* **⚡ AI Content Generation:** Integrates with an AI endpoint (via Dify) to automatically generate Question/Answer pairs based on user input.
* **🃏 3D Interactive UI:** Features a smooth, CSS-driven 180-degree 3D flip animation for a realistic flashcard experience.
* **🧠 Smart Array Management:** Utilizes Vanilla JavaScript to handle state management, dynamically updating the DOM as the user cycles through their customized deck.
* **🎯 Distraction-Free Design:** A minimalist, highly focused user interface built to maximize study efficiency.
